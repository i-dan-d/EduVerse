Thông tin arXiv API

📡 Endpoint API

https://export.arxiv.org/api/query

🔧 Tham số request

  • search_query: Từ khóa tìm kiếm
  • start: Vị trí bắt đầu (pagination)
  • max_results: Số kết quả tối đa
  • sortBy: Cách sắp xếp (relevance, lastUpdatedDate, submittedDate)
  • sortOrder: Thứ tự (ascending, descending)

📝 Định dạng dữ liệu

API trả về dữ liệu dạng XML/Atom feed với các thông tin:

  • Title: Tiêu đề bài báo
  • Authors: Tác giả
  • Abstract: Tóm tắt
  • Published: Ngày xuất bản
  • Categories: Danh mục chuyên ngành
  • PDF Link: Liên kết file PDF

🎓 Hiển thị cho học sinh

Dữ liệu được xử lý và hiển thị thân thiện với học sinh THCS/THPT:

  • Tóm tắt đơn giản hóa
  • Phân loại theo môn học
  • Gợi ý ứng dụng thực tế
  • Từ khóa và khái niệm chính
